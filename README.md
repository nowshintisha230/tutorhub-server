# 🚀 TutorHub Backend

🌐 Live API URL: https://tutorhub-server-plum.vercel.app


TutorHub Backend is a REST API server built with Express.js and MongoDB that powers the TutorHub platform. It handles tutor management, booking system, authentication, and database operations.

---

## ✨ Features

➤ 🧑‍🏫 Create, update, delete tutor listings  
➤ 🔍 Search tutors by name with case-insensitive filtering  
➤ 📅 Filter tutors using start and end date  
➤ 📌 Featured tutors API (limited results)  
➤ 🔐 Secure route protection using JWT authentication  
➤ 📚 Book tutoring sessions with slot validation  
➤ ❌ Cancel bookings and update status  
➤ ⚡ MongoDB Atlas cloud database integration  
➤ 🌐 CORS enabled for frontend connection  

---

## 🛠️ Tech Stack

- Node.js  
- Express.js  
- MongoDB Atlas  
- JWT (jose) Authentication  
- dotenv  
- cors
